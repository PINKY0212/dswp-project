Dataset Overview

We use the Open Power System Data (OPSD) Time Series dataset (Germany) as the primary source for electricity demand. Weather variables are aligned to the same hourly timestamps (used for LSTM / Hybrid models).

Open OPSD Dataset

What we use

  • Country: Germany
  • Resolution: 60 minutes
  • Target: Electricity load (MW)
  • Additional inputs: Calendar features + weather variables (for LSTM / Hybrid)

Notes on usage

  • All timestamps are standardized to an hourly time index.
  • Missing values are handled during preprocessing to ensure continuity.
  • Processed dataset is shared across SARIMA / LSTM / Hybrid / Ensemble for fair comparison.
